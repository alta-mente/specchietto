Underlined content tabs with optional icons and count pills. Controlled.

```jsx
<Tabs value={tab} onChange={setTab} tabs={[
  { id: 'oggi', label: 'Oggi', count: 38 },
  { id: 'settimana', label: 'Settimana', count: 204 },
  { id: 'ospiti', label: 'Ospiti', icon: <Icon name="users" size={16} /> },
]} />
```
