A small status pill. Use the reservation-status tones for booking states — they map to the status palette.

```jsx
<Badge tone="confirmed" dot>Confermata</Badge>
<Badge tone="pending" dot>In attesa</Badge>
<Badge tone="vip" dot>Ospite VIP</Badge>
<Badge tone="danger" variant="solid">Annullata</Badge>
```
Tones: confirmed·pending·seated·vip·cancelled·note (or semantic success/warning/danger/info/accent/special/neutral). Variants: soft (default), solid, outline.
